# Healthcare Data Analysis Project

## Overview
This project analyzes healthcare imaging workflow data to identify trends, bottlenecks, and improvement opportunities.

## Tools Used
- Python
- Pandas
- Matplotlib
- SQL (simulated)

## Key Insights
- Peak imaging times
- Average turnaround delays
- Resource utilization trends
